# Pygame_Platformer
A SIMPLE PLATFORMER GAME

# INSTRUCTION
<kbd>`WASD`</kbd> : Move <br />
<kbd>`J`</kbd> : Throw kunai <br />
<kbd>`L`</kbd> : Dash <br />

# REQUIREMENTS:
```PYTHON
PYGAME (2.4.0 or higher)
```
